# Dev Docs Prompts

Dev Docs is an app to chat with a direct knowledge base of developer documentation to get excatly what you want when you want it.

Each Markdownfile is an additive, some are standard, some depend on the stack space created. 

A stack space is the list of frameworks, sdk, etc. to reference.  
Categories that add special instructions include Frontend, Authentication, etc. 

The reference page has common developer documentation and common developer conceptual practices.
